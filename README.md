# Plateforme de Prédiction de Centralité Client

## Description
Projet de mémoire de fin d'étude - Master en Data Science
Plateforme d'analyse et de prédiction de la centralité des clients dans un réseau.

## Objectifs
- Analyser les réseaux de clients et leurs interactions
- Prédire la centralité des clients (degree, betweenness, closeness, eigenvector)
- Fournir des visualisations interactives
- Proposer des recommandations basées sur l'analyse

## Structure du Projet
```
client-centrality-prediction-platform/
├── data/               # Données brutes et traitées
├── notebooks/          # Notebooks Jupyter pour l'exploration
├── src/                # Code source principal
├── models/             # Modèles entraînés
├── tests/              # Tests unitaires
├── docs/               # Documentation
├── config/             # Fichiers de configuration
└── requirements.txt    # Dépendances Python
```

## Installation
```bash
pip install -r requirements.txt
```

## Utilisation
```bash
python main.py
```

## Technologies Utilisées
- Python 3.9+
- NetworkX (analyse de réseaux)
- Scikit-learn (machine learning)
- Pandas & NumPy (manipulation de données)
- Matplotlib & Seaborn (visualisation)
- Streamlit (interface web)
